# Mapty Website

The project created following the instructions from the [JavaScript course on Udemy](https://www.udemy.com/course/the-complete-javascript-course/?utm_campaign=website1010&utm_medium=website1010&utm_source=mycoupon) by Jonas Schmedtmann.

## Functionality

- loads a map centered on the current location
- with a click on the map we can create a workout
- at creation of the workout, we render a marker on the map, and store the workout in the local storage

## Things learned

- project planning: user stories
- formatting the script according to the OOP principles
- local storage API
- geolocation API

## Technologies

- leaflet 1.6.0
